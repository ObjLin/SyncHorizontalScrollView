# SyncHorizontalScrollView
横竖联动的ScrollView

##先上传张效果图

![image](https://github.com/chensong1993/SyncHorizontalScrollView/blob/master/newsAd.gif)

主要实现功能
竖列第一列只能上下滑动不能左右滑动
横列标题和内容可以左右联动
